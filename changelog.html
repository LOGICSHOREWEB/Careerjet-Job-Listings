Changelog: Careerjet Job Listings Pro

Version 2.2.1 - January 2025

Improvements:

  • Enhanced UI for Admin Settings: Simplified and improved the layout of the Careerjet settings page, making it more user-friendly for plugin administrators.
  • Job Listing Display Enhancements: Optimized how job listings are displayed, ensuring better formatting and clearer presentation of job details such as company name, location, salary, and application link.
  • API Integration Update: Updated the API integration with Careerjet, ensuring smoother data retrieval and enhanced compatibility with recent Careerjet API changes.
  • Security Enhancements: Improved the security of the plugin by adding better data sanitization to avoid potential vulnerabilities in displaying job listings.

Bug Fixes:

  • Sidebar Widget Fix: Resolved an issue where job listings were not displaying correctly in the sidebar widget when multiple keywords were used.
  • Location Display Fix: Fixed a bug where job locations were incorrectly displayed or missing in some cases. Now, location data is consistently shown.
  • Shortcode Issue: Fixed a problem where the shortcode [careerjet_jobs] was not rendering job listings correctly when used on some themes.
  • Job Listing Loop Fix: Corrected a looping issue that caused the plugin to fetch an excessive number of job listings, resulting in layout breaks or long loading times.

Version 2.2 - December 2024

New Features:

  • Customizable Display Locations: Added a new setting allowing administrators to choose between three job listing display options: after post content, in the sidebar widget, or using the [careerjet_jobs] shortcode.
  • Keywords Management: Introduced an option to add multiple keywords (separated by commas) to fetch a broader range of job listings from Careerjet, with configurable display count per keyword.
  • Job Display Limit: Administrators can now set a total number of job listings to display, as well as the maximum number per keyword, providing greater control over the number of results shown on the site.

Improvements:

  • Performance Optimization: Improved the performance of job listings fetching, reducing the load time and optimizing how data is displayed across different pages.
  • UI/UX Enhancements: Enhanced the user interface for managing plugin settings, making it more intuitive and easier to use.

Bug Fixes:

  • Job Listing Format Fix: Fixed an issue where job titles and other details were being improperly formatted, causing inconsistent display.
  • Widget Display Issue: Resolved a display issue that caused the widget to show blank content in certain themes.

Version 2.1 - October 2024

Improvements:

  • Job Details Display Update: Improved the display of job details (company, salary, location, and apply link) with better styling, ensuring a more visually appealing and user-friendly presentation.
  • Localization Update: Updated the plugin for better localization and language support, particularly for users in Canada and the U.S.

Bug Fixes:

  • Plugin Conflicts: Fixed compatibility issues with some other popular plugins, preventing conflicts that could disrupt the display of job listings on the site.
